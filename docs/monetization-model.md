# Modelo de Monetização - Próximo Passo

## Estratégia Geral

Freemium SaaS com foco em Premium por valor, não por restrições duras.

---

## 📊 Planos de Assinatura

### 1. PLANO GRATUITO
**Preço**: R$ 0/mês

**Objetivo**: Aquisição e validação de mercado

**Features:**
- ✓ Explorar 3 jornadas (sem salvar progresso)
- ✓ Blog completo (100% acesso)
- ✓ 3 ferramentas básicas (calculadora MEI, financiamento, IR)
- ✗ Sem acesso a especialista IA
- ✗ Sem checklist persistente
- ✗ Sem histórico
- ✗ Sem lembretes

**Conversão**: ~5-10% dos usuários free → Premium

**Estratégia**:
- Deixar experimentar o valor
- Cobrar quando quer salvar progresso ("Quer continuar essa jornada? Assine Premium")
- CTA estratégico no meio da jornada

---

### 2. PLANO PREMIUM
**Preço**: R$ 29/mês (ou R$ 290/ano com desconto de 15%)

**Objetivo**: Usuários que precisam de acompanhamento completo

**Features:**
- ✓ Jornadas ilimitadas
- ✓ Salvar progresso em tempo real
- ✓ Histórico completo (todas as ações)
- ✓ Acesso a especialista IA contextualizados
- ✓ Exportar checklist em PDF
- ✓ Sem anúncios
- ✓ Lembretes por email/push
- ✓ Acesso a documentos templates
- ✓ Timeline visual com prazos
- ✓ Calculadoras avançadas
- ✓ Suporte por email

**Conversão**: Meta 40-50% de usuários Premium

**Estratégia**:
- Focar em valor do especialista IA
- Ofertas: "3 primeiros meses por R$ 19/mês"
- Oferecer trial de 7 dias grátis

---

### 3. PLANO FAMÍLIA
**Preço**: R$ 59/mês (ou R$ 590/ano com desconto de 15%)

**Objetivo**: Casais, famílias inteiras resolvendo problemas juntos

**Features:**
- ✓ Tudo do Premium
- ✓ Até 5 pessoas na mesma conta
- ✓ Compartilhar progresso entre membros
- ✓ Dashboard compartilhado
- ✓ Atribuir tarefas entre membros
- ✓ Histórico compartilhado
- ✓ Suporte por email prioritário
- ✓ Relatórios para família

**Conversão**: Meta 15-20% de usuários Premium para Família

**Segmentação**:
- Casais comprando casa: "Acompanhem juntos"
- Filhos ajudando pais: "Organize as tarefas"
- Irmãos em inventário: "Coordene a herança"

### 4. PLANO EMPRESARIAL (B2B)
**Preço**: Desde R$ 499/mês (SKU custom)

**Objetivo**: Contadores, advogados, profissionais

**Features:**
- ✓ Tudo do Premium
- ✓ Múltiplos usuários/clientes
- ✓ API para integração com software próprio

- ✓ Dashboard de analytics avançado

**Segmentação**:

**Estratégia de Go-to-Market B2B**:
1. Validar Freemium + Premium primeiro

## 🎯 Estratégia de Conversão

### 1. Ponto de Parada Natural
```
Usuário explora 3 jornadas gratuitas
     ↓
Tenta salvar progresso na 4ª jornada
     ↓
Modal: "Salvar progresso é um recurso Premium"
     ↓
CTA: "Assinar Premium (7 dias grátis)"
```

### 2. Value Demonstration
- Mostrar especialista IA em ação (demo)
- Amostra de checklist salvo
- Case de usuário ("João economizou 10h com isso")

### 3. Ofertas Estratégicas
- **Novo usuário**: 7 dias grátis
- **Usuário hesitante**: "3 primeiros meses por R$ 19" (R$ 57 vs R$ 87)
- **Black Friday/Cyber Monday**: 50% desconto anual
- **Referral**: "Convide um amigo, ganhe 1 mês grátis"

### 4. Email Marketing
- **Onboarding**: "Veja o valor do Premium" (dia 2)
- **Hesitante**: "Você ainda não salvou nada" (dia 7)
- **Inativo**: "Suas jornadas estão aqui" (dia 14)

## 📈 Métricas de Monetização

| Métrica | Target | Fórmula |
|---------|--------|---------|
| CAC (Cost of Acquisition) | < R$ 50 | Spend / Conversões |
| LTV (Lifetime Value) | > R$ 500 | ARR / Churn |
| Churn Monthly | < 5% | (Saídas / Início) |
| ARR | R$ 100k+ | MRR × 12 |
| MRR | R$ 8k+ | Sum(Revenue) |
| NRR (Net Revenue Retention) | > 110% | MRR final / MRR início |
| Conversion Rate | 8-15% | Premium subscribers / Total users |

---

## 🛡️ Estratégia Anti-Churn

1. **Onboarding**: Primeira jornada completada = 90% menos churn
2. **Engagement**: Email semanal com progresso
3. **Support**: Resposta rápida a problemas
4. **Win-back**: "Volte e ganhe 30% de desconto"
5. **Premium to Familia**: Upsell automático quando adiciona segunda pessoa
